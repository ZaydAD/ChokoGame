# Choko Game

A strategic board game developed in Java using JavaFX, accompanied by a website for presentation and documentation.

---

## Collaboration

This project was developed as part of a group. I contributed to the design and implementation of the application, including core features, architecture, and overall functionality. The project reflects both collaborative teamwork and individual contributions.

---

## Overview

This project builds on my earlier Wordle ASCII game and represents a transition to JavaFX for graphical user interface development. It demonstrates improvements in code structure, usability, and overall design.

The application follows a layered architecture (Model-View-Presenter), ensuring clear separation of concerns and improved maintainability. During development, I also gained practical experience working with frameworks.

---

## Features

* Strategic board game implementation
* Rule-based AI (capture, blocking, win detection)
* JavaFX graphical user interface
* Layered architecture (MVP)
* Companion website for presentation
